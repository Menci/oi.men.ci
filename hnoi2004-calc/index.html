「HNOI2004」高精度开根 - 高精度 + 二分

发表于 2017-02-28 | 分类于 OI

给 和一个高精度数 ，求 。

链接

BZOJ 1213

题解

首先，从 开始每次扩大一倍，确定答案的范围。

然后二分答案检验即可。

高精度用 Python 比较方便。

代码

m = input()
n = input()
l = 0
r = 1
while r ** m < n:
    r *= 2

while l < r:
    # print "l = %d, r = %d" % (l, r)
    mid = (l + r) // 2 + 1
    x = mid ** m
    if x <= n:
        l = mid
    else:
        r = mid - 1
print l
# BZOJ # 高精度 # HNOI # 二分
